UcanHaz Reference Books:  A Codex Plug-in for Google Quick Search Box

Usage:

Just add any text into QSB (don't forget to prepend with a space) press "tab" to pivot to an action and select a codex to search.  Typing "codex" will bring up all available codex sources or you can just start to type the name of a specific source, e.g. WordPress, JQuery, and so on.  Once you have selected the codex to be searched press return.

Current codex sources include HTML, CSS, PHP, WordPress, MySQL, Python, and JQuery
